# 🚛 Andrade Transportes

## Informaões do Projeto

**Projeto:** Site Institucional Andrade Transportes  
**Status:** Em produção  
**Deploy:** andrade-transportes.vercel.app
**Desenvolvedor:** Lucas Oliveira de Deus  

A aplicação foi desenvolvida para apresentar institucionalmente a Andrade Transportes, empresa especializada em logística e transporte frigorificado, com foco em qualidade, segurança e controle rigoroso de temperatura.

---

### 🔹 Usando sua IDE

Caso queira trabalhar localmente:

#### Clone o repositório

```bash
git clone <URL_DO_REPOSITORIO>
```

---

## Primeiros Passos

Primeiro, instale as dependências do projeto:

```bash
npm install
# or
yarn install
```

Em seguida, inicie o servidor de desenvolvimento:

```bash
npm run dev
# or
yarn dev
```

Abra [http://localhost:8081/](http://localhost:8081/) no seu navegador para ver o resultado.

Você pode começar a editar a página modificando o arquivo app/page.tsx. A página será atualizada automaticamente conforme você edita o arquivo.

Este projeto usa [`vite/font`](https://vite.dev/guide/).
